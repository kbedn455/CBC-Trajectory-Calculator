# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] — 2025

### Added
- XYZ targeting mode — enter Cannon Mount and target coordinates to get yaw and pitch
- Manual mode — simulate trajectory and range for any cannon configuration
- Big Cartridge support with power levels 1–4 and stress display
- Powder Charge support with velocity and stress display
- Y correction button (−25 blocks) to reduce ricochet risk
- Warnings: cannon jamming, ricochet risk, Mortar Stone speed limit, no charges
- Maximum range display for current velocity
- Trajectory chart (parabola visualization)
- PL / EN language toggle
- Material limits: Log, Wrought Iron, Cast Iron, Bronze, Steel, Nethersteel
- All projectile types supported
